Ejercicio 2

Búsqueda de un PPA y comprobación de paquetes disponibles

Para este ejercicio se seleccionó el PPA oficial de LibreOffice como repositorio alternativo. El objetivo era identificar un PPA, consultarlo y verificar qué paquetes están disponibles en él. En este caso se utilizó Linux Mint como entorno de trabajo.

1. Acceso al PPA desde Launchpad

En primer lugar, se accedió al siguiente enlace donde se encuentra alojado el PPA de LibreOffice:

URL: https://launchpad.net/~libreoffice/+archive/ubuntu/ppa

En la página se puede observar el contenido del repositorio, incluyendo los paquetes y las distribuciones soportadas.

Captura 1: Visualización del PPA en Launchpad.

2. Añadir el PPA al sistema

En Linux Mint es posible añadir un PPA mediante el comando add-apt-repository. Para añadir el repositorio se ejecutó el siguiente comando:

sudo add-apt-repository ppa:libreoffice/ppa

Tras su ejecución, el sistema incorpora el repositorio al listado de orígenes de software.

Captura 2: Añadiendo el PPA desde la terminal.

3. Actualización de la caché de paquetes

Para que el sistema reconozca los paquetes ofrecidos por el PPA es necesario actualizar la caché de apt mediante el comando:

sudo apt update

Este comando descarga la información de los paquetes desde el repositorio recién añadido.

Captura 3: Actualización de la caché de APT.

4. Comprobación de paquetes disponibles

Finalmente, se realizó una búsqueda para listar los paquetes relacionados con LibreOffice disponibles desde el repositorio:

apt search libreoffice

La salida muestra múltiples paquetes como writer, calc, impress, help y otros módulos.

Captura 4: Listado de paquetes proporcionados por el PPA.

Conclusión

Se ha ubicado un PPA válido, se ha añadido al sistema y se ha comprobado que ofrece múltiples paquetes relacionados con LibreOffice. Con ello se cumple el objetivo del ejercicio.
