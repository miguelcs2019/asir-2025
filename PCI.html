Actividad 2 – Componentes PCI Express (PCIe)

1. Componentes por tamaño de ranura PCIe

A continuación se muestran ejemplos de componentes reales que se pueden encontrar en tiendas web, clasificados por el tamaño de ranura PCIe que utilizan (x1, x4, x8, x16). Se incluyen varios dispositivos en cada tamaño y una breve explicación de sus características más importantes.

Ranura PCIe x1

PCIe x1

Tarjeta de sonido dedicada

  • Ofrece mejor calidad de audio que el sonido integrado de la placa base.
  • Suele incluir salidas 5.1 o 7.1 y entrada para micrófono.
  • PCIe x1 es suficiente porque el ancho de banda que necesita el audio es bajo.
PCIe x1

Tarjeta de red WiFi + Bluetooth

  • Añade conectividad WiFi 5/6 y Bluetooth a un PC de sobremesa.
  • No requiere mucha velocidad, por eso se usa un conector pequeño x1.
  • Ideal para equipos que no tienen WiFi integrado en la placa base.

Ranura PCIe x4

PCIe x4

Tarjeta adaptadora NVMe (M.2 a PCIe)

  • Permite instalar un SSD M.2 NVMe en una ranura PCIe x4.
  • Aprovecha velocidades muy altas (hasta varios GB/s).
  • Se usa cuando la placa base tiene pocas ranuras M.2 integradas.
PCIe x4

Controladora RAID / almacenamiento

  • Añade varios puertos SATA o incluso puertos SAS para discos profesionales.
  • Necesita más ancho de banda que una tarjeta x1, por eso usa x4.
  • Permite crear arreglos RAID para mejorar velocidad o seguridad de los datos.

Ranura PCIe x8

PCIe x8

Tarjeta HBA / Controladora de servidor

  • Se usa en servidores para conectar muchos discos o cabinas de almacenamiento.
  • Necesita mucho ancho de banda para mover datos de varios discos a la vez.
  • PCIe x8 es un equilibrio entre rendimiento alto y número de ranuras disponibles.
PCIe x8

Tarjeta gráfica profesional de gama media

  • Algunas GPUs profesionales funcionan en ranuras x8 sin perder mucho rendimiento.
  • Se usan en estaciones de trabajo para CAD, diseño o edición de vídeo.
  • Permite dejar otra ranura x8 libre para más tarjetas de expansión.

Ranura PCIe x16

PCIe x16

Tarjeta gráfica gaming de alta gama

  • Necesita el máximo ancho de banda disponible para mover muchos datos de vídeo.
  • Se utiliza para juegos 3D, realidad virtual y edición de vídeo 4K/8K.
  • Por eso siempre se monta en la ranura principal PCIe x16 de la placa base.
PCIe x16

Tarjeta capturadora de vídeo 4K

  • Permite capturar vídeo 4K/60 fps de consolas u otros PCs.
  • El flujo de datos de vídeo es muy alto, por eso se usa x16.
  • Se utiliza en streaming profesional, grabación y edición de contenido.

2. Componentes por generación de PCIe

Aquí se recogen ejemplos de dispositivos que aprovechan distintas generaciones de PCIe (Gen3, Gen4, Gen5). Cada generación duplica aproximadamente el ancho de banda de la anterior, lo que permite crear dispositivos más rápidos.

PCIe Gen3

PCIe 3.0

SSD NVMe PCIe 3.0

  • Velocidades típicas: 3.000–3.500 MB/s de lectura.
  • Supuso un salto enorme frente a los SSD SATA (≈550 MB/s).
  • Sigue siendo suficiente para muchos PCs de uso general.
PCIe 3.0

Tarjeta gráfica PCIe 3.0

  • Muchas GPUs de hace unos años funcionan en PCIe 3.0 x16.
  • Ofrecen buen rendimiento en juegos 1080p y 1440p.
  • La interfaz no es el cuello de botella, lo es la propia GPU.

PCIe Gen4

PCIe 4.0

SSD NVMe PCIe 4.0

  • Velocidades típicas: 5.000–7.000 MB/s de lectura.
  • Reduce muchísimo los tiempos de carga en juegos y aplicaciones.
  • Ideal para edición de vídeo 4K y trabajo con archivos muy pesados.
PCIe 4.0

Tarjeta gráfica moderna PCIe 4.0

  • Las GPUs actuales de gama alta ya usan PCIe 4.0.
  • Permite mover más datos entre GPU y CPU, útil en resoluciones altas.
  • Está muy extendido en placas base de última generación.

PCIe Gen5

PCIe 5.0

SSD NVMe PCIe 5.0

  • Velocidades que superan los 10.000 MB/s de lectura.
  • Se utilizan disipadores grandes porque generan bastante calor.
  • Muy orientados a PCs de gama muy alta y estaciones de trabajo.
PCIe 5.0

Tarjeta gráfica preparada para PCIe 5.0

  • Algunas GPUs nuevas están empezando a usar PCIe 5.0.
  • El ancho de banda extra será útil en aplicaciones profesionales y 8K.
  • Requiere placas base y procesadores compatibles con esta generación.

3. ¿Por qué PCIe sigue evolucionando?

PCIe continúa evolucionando porque cada vez necesitamos mover más datos entre los componentes del ordenador. Los procesadores tienen más núcleos, las tarjetas gráficas procesan más información para juegos y trabajos en 4K/8K, y los SSD NVMe son cada vez más rápidos. Si la interfaz PCIe se quedara estancada, se convertiría en un cuello de botella.

Cada nueva generación de PCIe (Gen3, Gen4, Gen5...) mejora el ancho de banda disponible. Esto permite:

  • Cargar juegos y programas mucho más rápido.
  • Trabajar con vídeo y modelos 3D muy pesados sin tantos tiempos de espera.
  • Conectar más dispositivos al mismo tiempo sin perder rendimiento.
  • Preparar los PCs para futuras tarjetas gráficas y SSD aún más potentes.

PCIe sigue evolucionando porque el hardware no deja de mejorar y la interfaz debe ser capaz de seguir el ritmo para que todo el sistema funcione sin cuellos de botella.

En este apartado se comparan dos dispositivos reales de fabricantes distintos para cada tipo de ranura PCIe. Así se puede ver que, aunque usen el mismo conector, pueden estar orientados a usos diferentes.

PCIe x1 – Tarjetas de sonido

Creative

Sound Blaster Audigy FX

  • Tarjeta de sonido PCIe x1.
  • Mejora clara frente al audio integrado.
  • Enfocada a usuarios domésticos y gaming.
ASUS

Xonar SE

  • Tarjeta de sonido PCIe x1.
  • Mejor aislamiento y calidad de audio.
  • Muy usada para música y multimedia.

PCIe x4 – Almacenamiento NVMe

Samsung

Samsung 980 PRO

  • SSD NVMe PCIe 4.0 x4.
  • Velocidades muy altas de lectura y escritura.
  • Ideal para gaming y edición.
WD

WD Black SN850

  • SSD NVMe PCIe 4.0 x4.
  • Rendimiento similar al Samsung.
  • Muy popular en PCs de alto rendimiento.

PCIe x8 – Tarjetas de red profesionales

Intel

Intel X550-T2

  • Tarjeta de red PCIe x8.
  • Conectividad 10 Gb Ethernet.
  • Muy usada en servidores.
Broadcom

NetXtreme BCM57412

  • Tarjeta de red PCIe x8.
  • Enfoque profesional y empresarial.
  • Alta fiabilidad y rendimiento.

PCIe x16 – Tarjetas gráficas

NVIDIA

GeForce RTX 4070

  • Tarjeta gráfica PCIe x16.
  • Muy orientada a juegos y creación de contenido.
  • Uso doméstico y gaming avanzado.
AMD

Radeon RX 7800 XT

  • Tarjeta gráfica PCIe x16.
