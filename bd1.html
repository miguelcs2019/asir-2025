UD07 – Tarea 1: Instalación Ubuntu Server

Objetivo de la práctica

En esta práctica se realiza la instalación de Ubuntu Server, que servirá como base para montar un sistema de bases de datos completo (Apache + MySQL + PHP).

El objetivo es dejar el sistema preparado, limpio y correctamente configurado para su uso como servidor.

Proceso de instalación paso a paso

1. Creación de la máquina virtual

Se crea una máquina virtual asignando:

  • RAM: mínimo 2 GB (recomendado 4 GB)
  • Disco: mínimo 20 GB
  • CPU: 2 núcleos si es posible

Se monta la ISO de Ubuntu Server como dispositivo de arranque.

2. Inicio del instalador

Al arrancar la máquina se selecciona:

Install Ubuntu Server

El resto de opciones NO se utilizan en este caso.

3. Idioma, teclado y red

Idioma: seleccionar el deseado (ej: Español).

Teclado: dejar automático o seleccionar español.

Red:

  • Se configura automáticamente (DHCP).
  • NO modificar si no sabes lo que haces.

4. Configuración del proxy

Este paso se deja vacío.

RECOMENDADO: no configurar proxy salvo que se requiera.

5. Configuración de repositorios

Se puede dejar el repositorio por defecto.

RECOMENDADO: no cambiarlo.

6. Particionado del disco

Seleccionar:

Use entire disk

  • Modo automático
  • No complicarse con particiones manuales

IMPORTANTE: confirmar los cambios cuando lo pida.

7. Creación de usuario

Se introduce:

  • Nombre del usuario
  • Nombre del servidor
  • Contraseña

Este usuario será el administrador del sistema.

RECOMENDADO: usar una contraseña segura.

8. Instalación de OpenSSH

Aquí aparece una opción importante:

Install OpenSSH server

RECOMENDADO: ACTIVAR esta opción ✔

  • Permite conectarse remotamente al servidor
  • Muy útil para administración

9. Selección de paquetes adicionales

Aparece una lista de servicios opcionales.

IMPORTANTE:

  • NO seleccionar nada (instalación limpia)
  • O como mucho OpenSSH si no se seleccionó antes

Ejemplos de lo que NO instalar ahora:

  • Docker
  • Kubernetes
  • Mail server
  • Otros servicios

RECOMENDACIÓN CLAVE:

Instalar solo lo necesario más adelante manualmente para evitar errores y mantener el sistema limpio.

10. Finalización

Se completa la instalación y el sistema se reinicia.

Después del reinicio:

  • Retirar la ISO
  • Iniciar sesión con el usuario creado

Conclusión

La instalación de Ubuntu Server se ha realizado correctamente siguiendo una configuración básica y limpia.

Se ha priorizado:

  • ✔ Simplicidad
  • ✔ Seguridad
  • ✔ Control del sistema

Este entorno queda preparado para la instalación posterior de servicios como Apache, MySQL y PHP, fundamentales en sistemas de bases de datos.
